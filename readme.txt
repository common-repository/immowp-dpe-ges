== Description ==

Afficher le rapport énergétique DPE et GES d'un bien immobilier sur votre site. Choisir d'afficher les estimations de coûts annuels, les géorisques (2023).

Le plugin ImmoWP DPE GES vous offre la possibilité de générer des indicateurs de consommation énergétique, d'émission de gaz et d'estimation des coûts annuels. 
Il est facile à utiliser, intuitif et permet de saisir ces informations dans des types de publications personnalisés. 
Il affiche une boîte de méta DPE dans les publications et les pages WordPress et est compatible avec les dernières versions de WordPress. 
Différentes options permettent de prendre en charge les biens en zone climatiques rigoureuses, les biens non soumis au DPE ainsi que les géorisques.
Ce plugin est idéal pour les thèmes immobiliers et est compatible avec tous les types de passerelles immobilières.


== Installation ==

= Méthode Administration WordPress =
1. Allez dans votre administration de Wordpress `Extensions > Ajouter`
2. Rechercher `ImmoWP DPE-GES` (utiliser le formulaire de recherche)
3. Cliquez sur le bouton Installer et activer de l'extension

= Méthode FTP =
1. Téléchargez le dossier complet `immowpdpeges` dans le répertoire `/wp-content/plugins/`
2. Activez l'extension depuis le menu 'Extension' dans WordPress
